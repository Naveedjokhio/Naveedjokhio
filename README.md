### 🏏 Cricbuzz Real-Time Data Pipeline (Flagship Project)

<img src="https://capsule-render.vercel.app/api?type=rect&color=0:16a085,100:2ecc71&height=4&width=100%25" width="100%"/>

Full GCP-based real-time streaming pipeline delivering live match data end-to-end, with a synthetic match simulator built in as a resilient fallback layer.

- ⚡ **Kafka → GCS → dlt → BigQuery → dbt** medallion flow, orchestrated with **Bruin + Kestra**

- 🎯 Built a synthetic match simulator as a fallback when the live RapidAPI quota ran out — keeping the pipeline demo-ready end-to-end
- 📊 Presented with a 20-slide deck and verified pipeline metrics
